# Images

Place your `image.jpg` file in this directory.

This is your profile/contact photo that will be displayed on the Contact section.

The image should be:
- Square format (1:1 aspect ratio recommended)
- At least 512x512px
- JPG, PNG, or WebP format
- Optimized for web (compressed)

## Example Structure

\`\`\`
public/images/
└── image.jpg
\`\`\`
