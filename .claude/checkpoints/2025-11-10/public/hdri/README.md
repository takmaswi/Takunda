# HDRI Files

Place your `studio_small_08_4k.exr` file in this directory.

The HDRI should be:
- In EXR format (for HDR support)
- 4K resolution recommended
- From Poly Haven (Studio Small 08)

## Download

You can download the HDRI from:
https://polyhaven.com/a/studio_small_08

Download the 4K EXR version.
